# Hibernate CRUD Application

## 📌 Project Overview

This project demonstrates basic CRUD (Create, Read, Update, Delete) operations using **Hibernate ORM** with a MySQL database. It shows how Hibernate simplifies database interaction by mapping Java objects to database tables.

---

## 🚀 Technologies Used

* Java
* Hibernate ORM
* MySQL
* Maven
* Eclipse IDE


---

## ⚙️ Features Implemented

* Insert Record
* Fetch Record by ID
* Update Record
* Delete Record
* HQL Queries
* Annotation-based Mapping

---

## 🗄️ Database Configuration

Update your `hibernate.cfg.xml` file with your database details:

```xml
<property name="hibernate.connection.url">jdbc:mysql://localhost:3306/your_database</property>
<property name="hibernate.connection.username">root</property>
<property name="hibernate.connection.password">your_password</property>
```

Make sure:

* MySQL server is running
* Database is created before running the project

---

## ▶️ How to Run the Project

1. Clone the repository
2. Import the project into Eclipse
3. Update database credentials
4. Run the main class
